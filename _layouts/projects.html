Notable Projects

  • gensend.email

    GenSend is a tool for generating and sending emails using AI.

    A demo is available at gensend.email

    This tool allows users to adjust parameters like formality, empowering users to generate emails that precisely match their desired tone and ensure effective communication in a variety of contexts.

    GenSend is powered by the OpenAI API. The interface is built in React, the and back end uses Node.js.

  • bike-safe-dashboard

    This is a data visualization project examining trends in cyclist and pedestrian safety in NYC.

    Demo | Source (Dashboard) | Source (API/data)

    The project's back end employs Python, utilizing Pandas DataFrames to extract and categorize pertinent details from a sizable dataset of around 1.9 million rows, sourced from NYC Open Data.

    Its focus is on identifying and classifying information related to accidents involving cyclists and/or pedestrians.

    A custom API was implemented to serve this data to a React front end, displaying interactive charts that showcase trends in cyclist and pedestrian safety over time.

    The live site continually updates its statistics on a daily basis.

  • FortySix

    FortySix is a browser-based distributed computing platform.

    Source

    FortySix allows users to customize and run genetic algorithms and distribute the computational load across multiple browser clients.

    Anyone can contribute CPU resources to an ongoing computation by opening a URL in their browser.
